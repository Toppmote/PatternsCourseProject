Курсовая работа

"Проектирование системы выявления вредоносного поведения людей по действиям в социальных сетях"

Студент Чупиков П. В. Группа ПИ-81

В данной курсовой работе была реализована имитационная модель системы выявления вредоносного поведения людей по действиям в социальных сетях.

Работа имитационной модели основывается на случайной генерации пользователей социальной сети, для каждого из которых также генерируются случайные действия. Генерируются следующие действия: добавить нового друга, добавить новый пост, отправить сообщение и подтвердить свой аккаунт. После добавления нового поста или отправки сообщения текст данных действий проверяется фильтром системы.

Фильтр системы способен вычислять результаты фильтрации с помощью двух алгоритмов:
1. Стандартный алгоритм - блокирует пользователя тогда, когда процент вредоносных слов в тексте более 20
2. Строгий алгоритм - блокирует пользователя тогда, когда в тексте есть хотя бы одно вредоносное слово.
Алгоритм системного фильтра возможно поменять для большей наглядности работы системы.

Клиент может посмотреть состояние каждого пользователя, сгенерировать данные заново, сменить алгоритм работы системного фильтра, посмотреть результаты фильтрации для каждого пользователя.
Также имеется возможность поменять параметры генерации пользователей и их действий через файл проекта resources/config.properties.
